# Petnix 🥚🐣

Telegram Mini App — игра с питомцем в стиле Tamagotchi.

## Структура проекта

```
petnix/
├── bot/
│   ├── bot.py           # Telegram бот (aiogram 3)
│   └── requirements.txt
└── miniapp/
    ├── index.html       # Мини-приложение
    ├── style.css
    └── app.js
```

## Игровой процесс

| Фаза | Что происходит |
|------|---------------|
| 🥚 Яйцо | Появляется яйцо, обратный отсчёт 24 часа |
| 🐣 Питомец | Вылупляется питомец, за ним нужно ухаживать |
| 😵 Смерть | Если голод упал до 0 — питомец умирает |

### Статы питомца

- 🍖 **Сытость** — падает со временем; действие «Покормить» (+30)
- 😊 **Настроение** — падает медленнее; действие «Играть» (+25)
- ⚡ **Энергия** — падает от игры; действие «Спать» восстанавливает

### Развитие питомца

| Возраст | Вид |
|---------|-----|
| 0–24 ч  | 🐣 Птенец |
| 24–72 ч | 🐥 Цыплёнок |
| 3–7 дн  | 🐓 Петух |
| 7+ дн   | 🦅 Орёл |

## Запуск

### 1. Бот

```bash
cd bot
pip install -r requirements.txt
```

Создай `.env` (или вставь напрямую в `bot.py`):
```
BOT_TOKEN=your_token_from_botfather
MINIAPP_URL=https://your-domain.com
```

```bash
python bot.py
```

### 2. Мини-приложение

Папку `miniapp/` нужно раздавать через **HTTPS**-сервер
(Telegram требует HTTPS для Web Apps).

**Вариант A — GitHub Pages (бесплатно):**
1. Залей `miniapp/` в репозиторий
2. Включи GitHub Pages → ветка `main` → папка `/miniapp`
3. Скопируй URL и вставь в `bot.py` → `MINIAPP_URL`

**Вариант B — локальная разработка (ngrok):**
```bash
# в одном терминале — любой статический сервер
cd miniapp && python3 -m http.server 8080

# в другом — ngrok
ngrok http 8080
```
Скопируй HTTPS-URL из ngrok и вставь в `MINIAPP_URL`.

**Вариант C — Vercel / Netlify:**
Просто перетащи папку `miniapp/` в Vercel/Netlify.

### 3. Настройка бота в @BotFather

```
/newbot          → создай бота, получи токен
/setmenubutton   → установи кнопку меню с URL мини-приложения
```

## Состояние игры

Все данные хранятся в `localStorage` браузера Telegram.
Питомец «живёт» даже когда приложение закрыто — статы продолжают падать.
